http post请求传参

问题：

传对象数组格式参数，JSON.stringify 后，整个对象变成 key，value 为空

问题产生原因：

请求头 Content-Type 为
application/x-www-form-urlencoded，传参方式：key : value，整个 JSON 字符串当做 key。

解决方法：

用第三方插件 qs 处理，qs.stringify()序列化
